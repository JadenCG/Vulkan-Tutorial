# GLM (Version 1.0.3)

## Project Setup

To prepare this project for compilation, place all of the contents of GLFW into this folder.
For reference, the directory's layout should appear as the following:

```
glfw (This folder)
├─ cmake/
├─ docs/
├─ glm/
├─ test/
├─ util/
├─── .gitignore (From this repo)
├─── CMakeLists.txt
├─── copying.txt
├─── manual.md
└─── readme.md
```
